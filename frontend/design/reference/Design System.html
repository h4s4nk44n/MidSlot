Principles

Quiet surfaces, precise information.

Nothing in a clinical scheduler should shout. We lead with type and tabular rhythm; color is reserved for status and action. Every card is labelled like a chart slip. Data reads first, chrome second.

02 / Type

Typography

Newsreader (display) lends editorial weight and medical-journal legitimacy to titles. Geist (sans) handles all UI. Geist Mono marks data — timestamps, IDs, medication dosages — so numbers stay aligned and scannable. Three families, strict hierarchy.
